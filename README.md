# Blog & Portfólio Pessoal

Bem-vindo ao repositório do meu site pessoal. Este projeto serve como um hub central para meus artigos técnicos, portfólio de projetos e ferramentas úteis.

**Acesse o site:** [https://moacirjr10.github.io/MyProject/](https://moacirjr10.github.io/MyProject/)

---

## O que você vai encontrar

O site está organizado em quatro seções principais:

### 1. Blog (Home)
Um feed de notícias e artigos técnicos sobre desenvolvimento, servidores, IoT e automação. Aqui compartilho meus aprendizados e tutoriais passo a passo.

### 2. Sobre Mim
Uma página dedicada ao meu perfil profissional, onde detalho minha formação em Engenharia de Computação, minhas habilidades técnicas (Java, Node.js, C++, SQL) e meus contatos.

### 3. Apps (Ferramentas)
Uma coleção de utilitários práticos desenvolvidos por mim:
- **Calculadoras de Engenharia:** Para fumigação em blocos e silos.
- **Calculadora de Resistores:** Ferramenta interativa para leitura de códigos de cores.
- **Conversor de Unidades:** Metros para cm, km e milhas.

### 4. Projetos (Detalhes)
Páginas dedicadas a explicar profundamente cada projeto do meu portfólio, com galerias de fotos, explicações técnicas e links para os repositórios de código.

---

##  Estrutura Técnica Simplificada

O projeto foi construído com foco em simplicidade e performance:

- **Frontend:** HTML, CSS e JavaScript puros (sem frameworks pesados).
- **Conteúdo Dinâmico:** Os posts são carregados a partir de um arquivo JSON, facilitando a atualização sem mexer no código.
- **Comentários:** Sistema próprio de comentários com login social, rodando em um servidor seguro.

---

## Privacidade e Segurança

Este projeto segue boas práticas de segurança:
- Todo o tráfego é criptografado (HTTPS).
- Nenhuma informação sensível (senhas, chaves de API) é exposta no código público.
- O sistema de comentários possui proteção contra spam e abuso.
