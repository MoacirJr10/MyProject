Este projeto foi desenvolvido para otimizar o trabalho de fumigação em silos agrícolas. Utilizando HTML, CSS e JavaScript, criei uma ferramenta que calcula automaticamente a dosagem de pastilhas e sachês baseada no volume do silo.

O sistema também conta com um módulo de histórico local para registrar as aplicações realizadas.

### Funcionalidades
* Cálculo automático de volume (Blocos e Silos cilíndricos).
* Dosagem personalizada por tipo de grão (Soja, Milho, Arroz).
* Histórico salvo no navegador (LocalStorage).
